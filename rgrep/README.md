## 依赖的软件包
### anyhow = "1"
> **基于错误类型的接口对象，用于RUST程序中的常用错误处理**
### clap = "4.3.0"
> **命令行参数解析器**
### colored = "2"
> **终端颜色渲染**
### glob = "0.3"
> **文件路径的模式匹配**
### itertools = "0.10"
> **实现各种类型的迭代器（无限迭代器，有限迭代器和组合迭代器）**
### rayon = "1"
> **轻量级的数据并行库**
### regex = "1"
> **轻量级数据并行计算库，无数据竞争**
### thiserror = "1"
> **为标准库std::error::Error接口提供了一个方便的派生宏**
